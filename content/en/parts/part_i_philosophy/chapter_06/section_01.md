            <h4>Reincarnation of Roots</h4>
            <p>According to Rabbi Isaac Luria (the <em>Arizal</em>), the 600,000 root souls of humanity recur throughout history to be sifted and refined. The generation of the Flood returned at the Tower of Babel, during the destruction of the First Temple, and again during the Holocaust. Each life is a new opportunity to correct the "deficits" of the past.</p>
            <div class="concept-box">
            <strong>Job and Terach</strong>
            Job was the reincarnation of Terach, Abraham‚Äôs father. His suffering was not merely random; it was a process of purification.
            <ul>
            <li><strong>The Mistake:</strong> Job brought burnt offerings (spiritual) but neglected the peace offerings (which feed the poor).</li>
            <li><strong>The Lesson:</strong> Being religious is not the same as being righteous. True righteousness requires active kindness toward the vulnerable.</li>
            </ul>
            </div>
            <h4>The Ignorance of Free Will</h4>
            <p>If we knew all the facts and the exact consequences of our actions, we would have no <strong>Free Will</strong>‚Äîwe would be like clay in the hands of an artisan. To reveal our true character, God hides the inner workings of the universe. We must decide how to act based on trust and inner integrity, not just the certainty of reward.</p>
            <div class="concept-box">
            <strong>Chaos and Probability</strong>
            Big things in our lives often happen because of very small variations. This is the world of <strong>Chaos</strong>. But as we integrate these events, they average out into a <strong>Binomial Function</strong>‚Äîfarther from the intense light of the origin, but closer to the "vessels" that can hold the peace of understanding.
            </div>

<script>
function createStars() {
    const field = document.getElementById('star-field');
    field.innerHTML = '';
    for(let i=0; i<50; i++) {
        const star = document.createElement('div');
        star.className = 'star';
        star.style.left = Math.random() * 100 + '%';
        star.style.top = Math.random() * 100 + '%';
        const size = Math.random() * 3 + 1;
        star.style.width = size + 'px';
        star.style.height = size + 'px';
        field.appendChild(star);
    }
}

function updateRedShift(val) {
    const stars = document.querySelectorAll('.star');
    const text = document.getElementById('shift-text');
    
    // 0 = Blue (Approaching), 100 = Red (Receding)
    // Center is 50
    
    let color, speed, message;
    
    if (val < 45) {
        // Blue Shift
        const intensity = (50 - val) / 50; // 0 to 1
        color = `rgb(${255 * (1-intensity)}, ${255 * (1-intensity)}, 255)`; // Whiite to Blueish? No, making it bluer
        // Actually lets tint it blue
        // White is 255,255,255. Blue is 100, 150, 255.
        
        stars.forEach(s => {
            s.style.boxShadow = `0 0 ${intensity * 10}px #3b82f6`;
            s.style.background = '#bfdbfe';
        });
        message = "<strong>Blue Shift (Approaching):</strong> Revelation. The light interacts directly. Clarity is high, but the 'vessel' of faith is less necessary.";
    } else if (val > 55) {
        // Red Shift
        const intensity = (val - 50) / 50; 
        
        stars.forEach(s => {
             s.style.boxShadow = `0 0 ${intensity * 10}px #ef4444`;
             s.style.background = '#fecaca';
        });
        message = "<strong>Red Shift (Receding):</strong> Concealment. The source moves away to create space. This 'darkness' builds the vessel of Yearning. Great growth happens here.";
    } else {
        stars.forEach(s => {
            s.style.boxShadow = 'none';
            s.style.background = 'white';
        });
        message = "<strong>Equilibrium:</strong> A state of rest. Neither intense revelation nor deep concealment.";
    }
    
    text.innerHTML = message;
}

createStars();
